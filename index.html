הוראות הגדרה ראשונה
  1. פתח את Google Cloud Console
  2. צור פרויקט → APIs & Services → Enable APIs → Google Calendar API
  3. Credentials → Create Credentials → OAuth 2.0 Client ID
  4. בחר Web application
  5. הוסף Authorized JS origins:
    http://localhost:8080
    https://muaddibbb.github.io
  6. הדבק את ה-Client ID למעלה ולחץ התחבר

להרצה מקומית:
python3 -m http.server 8080
ואז פתח http://localhost:8080
